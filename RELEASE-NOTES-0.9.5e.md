# Character Builder 0.9.5e — Metamagic Replacement Detail Cards

This focused interface patch improves the approved custom Sorcerer Metamagic replacement flow without changing any progression rules.

## Replacement details

Below `Optional Metamagic Replacement`, Character Builder now displays source-native detail cards for both sides of the pending replacement.

The left card shows the Metamagic selected in `Replace`. The right card shows the Metamagic selected in `With`. Each card includes the official artwork, name, enabled source label, and enriched source description. Clicking a card opens the official source document.

The cards update immediately as the dropdowns change. Selecting only the removal side shows only the left card. Returning `Replace` to `No replacement` clears the dependent `With` value and removes the complete detail area.

## Scope

This release does not change Metamagic counts, replacement eligibility, duplicate validation, Item creation/removal, Advancement records, rollback, Warlock behavior, native Advancement, other classes, protected commits, Settings, or GM progression tools.
