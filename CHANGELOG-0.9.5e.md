# Character Builder 0.9.5e Changelog

## Added

- Added a detail-card area directly below `Optional Metamagic Replacement`.
- Added a removal card that displays the selected owned Metamagic image, source, and enriched description.
- Added a replacement card that displays the selected new Metamagic image, source, and enriched description.
- Added immediate card updates when either dropdown changes.
- Added source-document opening from either detail card.
- Added responsive one-column behavior on narrow windows.

## Unchanged

- Sorcerer Metamagic acquisition and replacement rules from 0.9.5c.
- Duplicate prevention and cascading replacement cleanup.
- Native ItemChoice Advancement values and embedded Item creation/removal.
- Warlock projected cantrip behavior and the 0.9.51 target-rebind hotfix.
- Other class progression, protected commits, Settings, and GM progression tools.
